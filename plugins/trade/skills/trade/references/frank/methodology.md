## 0. 一句话总纲

他的交易本质可概括为两条：**一，自己绝不爆仓；二，识别市场的脆弱结构（尤其"傻钱"的机械运行规律），
在它们被迫运动时反向获利。**

由此引出核心命题：支撑/阻力**不是画出来的，是受迫资金流（forced flows）的触发位**。一个价位有效，
当且仅当有某类资金在那里被迫买或被迫卖——dealer gamma 对冲、回购盘、CTA、vol control、
杠杆 ETF 再平衡、被动指数盘。没有受迫流锚定的线（如纯斐波那契图形派）他一概不看。

这套认识论即**内生市场理论（Endogenous Market Theory）**：大行情看似由 news 触发，
实质由**仓位结构、杠杆分布、波动率传染、被动资金的机械行为**共同决定——事件只是引线，不是炸药。

## 1. 指数级别：每周一套 GEX/DEX 关键位

每周一开盘前从 GEX dashboard 取一组结构位（个股结构看 Unusual Whales 一类工具），整周围绕它执行。
固定模板：

| 项目 | 含义 | 用法 |
|---|---|---|
| Spot | 盘前期货位置 | 相对各墙的距离决定本周脚本 |
| **HVL（高波动临界）** | 正/负 gamma 分水岭 | **轴心。上方=正 gamma 稳定区；下方=负 gamma 放大区** |
| Call Resistance（全期限） | 结构性天花板 | 远端目标，Spot 离得远时不参与日内判断 |
| Call Resistance（0DTE） | 日内第一道压力 | 反弹第一目标/止盈位；与全期限重合 → pinning |
| Put Support（结构性/全期限） | 远端防线 | 负 gamma 环境里是"减速带"，不是无脑抄底位 |
| Put Support（0DTE） | 最近的短线防守位 | 守住→反抽；跌破→看下一档 |
| Total GEX/DEX ± 1D 变化 | 结构强化还是退化 | GEX 转负 + 单日大失血 = 结构退化警报 |
| IV30 / IV Rank / HV30 | 波动环境 | 决定用什么结构表达观点（参考 pitfall 19） |

### HVL 规则（核心中的核心）

HVL = High Volatility Level，做市商 gamma 由正翻负的临界价。它**不是方向预测器，是路径/波动率预测器**——
决定"会不会出现自由落体"，而不是"涨还是跌"。

- **Spot 在 HVL 上方 = 正 gamma 区**：dealer 涨了卖、跌了买 → 波动被压制、回调被吸收，
  正 gamma 下很难出现丝滑的单边暴跌。这里**做多回调、卖波动**更顺。
- **Spot 在 HVL 下方 = 负 gamma 区**：dealer 越跌越卖、越涨越追 → 趋势自我强化、假突破频发；
  只有跌破 HVL 才具备自由落体的条件。这里**顺势、买波动、不要抄底**。
- **实战范式**：指数失守 HVL 的一瞬间，正 gamma 翻负、做市商被迫顺势卖，
  对应的动作是在**线下方加重空头**（顺势，不是抄底）。
- **反弹按 level 分段推演**：负 gamma 里反弹也很急，但路径是
  `Spot → HVL → 0DTE Call Wall → 全期限 Call Wall` 的分段修复，不是直接顺滑上攻；
  没站上 0DTE 墙之前，不谈更高的全期限墙。
- **两墙重合看 pinning**：当 0DTE 与全期限 Call Resistance 同价，价格容易在该位反复 pin、高位震荡消化。

### VIX 决定看 gamma 还是 vanna（重要补充）

经验法则：**VIX 高时以 vanna 为主、VIX 低时以 gamma 为主。**

- **低 VIX**：dealer gamma 主导 → 用 HVL/GEX 那套（上面的规则）。
- **高 VIX**：vanna/charm 主导 → 找 **vanna 支撑位**（vol 回落时 dealer 被迫买回的位）；
  高波动里的 squeeze 往往由 vanna 驱动，而非 gamma。
- **VWAP 与 gamma flip 是两个东西**，但常重合；日内可看日线 VWAP，收不上 VWAP 的 0DTE 多单就走。
- 负 gamma 环境容易上下双向扫，**开盘不要追空**。

### GEX/DEX 结构退化的读法

- Total GEX 仍正 = 舒服的压波动结构；**GEX 转负 = 进入易暴涨暴跌区**。
- 即使 GEX 仍正，但 **GEX/DEX 边际变化（1D Change）为负** = 高位结构在退化，
  进入"高位消化 + Call Wall 博弈"阶段，不是无脑强化。
- DEX 仍正而 GEX 转负 = 中长期仓位没转空、但短期 gamma 缓冲没了 → 最危险的"加杠杆顶部"。

## 2. 个股级别：GEX 位 + 龙气 + 少量经典图形位

个股的关键位要**提前标在图上、记进脑子**，来源有三（按权重）：

1. **个股期权 GEX 位**——同指数逻辑，call wall / put wall（如某热门股的最大 call wall）。
2. **龙气 / 独特支撑位**——龙头股有自己被资金记住的位（前高、突破回踩位）；突破后回踩前高常是好机会。
3. **经典图形位**——前高、颈线、趋势线、整数关口，但**仅作辅助**，朴素使用，无指标玄学。

> ⚠️ 杠杆 ETF（SOXL/KORU 等 2x/3x）**不能用正股那套看支撑**：它们因每日再平衡的衰减天然下移，
> 只适合"低买高卖"快进快出，止损必须挂在**标的指数**上、而非 ETF 自身价格。

## 3. 选股：相对强弱（RS）+ 龙气

Level 决定**何时打**，相对强弱决定**打什么**。

- **RS 定义**：跌时跌得少、涨时涨得多。多数个股日内只是大盘的相似图像；
  那些"跌得少、涨得多"的，才是判断 RS 的关键。
- **选股范式**：当板块（如 SMH）暴跌而某几只逆势不跌，这几只就是最强的龙头候选。
- **龙气**：最强的那一只就是龙王，回调时优先重点抄底它。

## 4. 受迫资金流玩家清单（"傻钱的运行规律"）

识别"谁在某个位置被迫动"，那个位置就是 level。标准清单：

| 玩家 | 机械规律 | 可利用的位置/时机 |
|---|---|---|
| **做市商（dealer）** | 按 gamma 对冲 | HVL、call/put wall —— 日内 gamma 支撑阻力的来源 |
| **CTA** | follow trend | 趋势确立后追；近月 CTA 转空 → 不怕被轧空 |
| **Vol control fund** | 跟踪 RV：RV↓加仓、RV↑减仓 | VIX 急升 → 被迫平高 beta（半导体小票），机械抛售 |
| **Dispersion 盘** | short 指数 vol、long 个股 vol | 管小票；赚 correlation premium；解除时小票剧烈 |
| **杠杆 ETF** | 每日按 NAV 再平衡 | 指数每动 1% → 上百亿美元级被动盘 |
| **回购盘** | 在 VWAP 附近承接 | 大回购公司回踩 VWAP 时有承接 → 可做 0DTE 进场 |
| **被动/指数盘** | 权重再平衡 | 单一持股上限触发的强制再平衡（如韩国 10% 上限案例）|

**VWAP = 回购盘的执行位**：知道是谁在 VWAP 买，所以敢在大回购公司回踩 VWAP 时用 0DTE call 进场。

**仓位结构 = 脆弱性信号**（内生理论的应用）：
- 一边 call-heavy + 杠杆拉满 = 上方脆弱（轧多）；多头仓位拥挤时，利空的反而是多头。
- risk reversal 从 call-skew 翻 put-skew = capitulation tell，可在另一侧建仓（如 MU complex）。
- COR1M、单股 vol/VIX 溢价处于历史极值 = 均值回归在即。
- 一句话：谁仓位拥挤，受迫平仓时就先爆谁。

## 5. Level 的执行规则（与"怎么找"同样重要）

1. **每条线绑定一个动作，二元化执行**：如"X 不破全力做多"、"跌破 A 减仓、跌破 B 清仓"。线就是 invalidation。
2. **先定线再定仓位**：止损越远，初始仓越小；止损越近，仓位可越大。
3. **卡边缘做不对称赌注**：贴线进场、止损极近 → 错了基本不亏、对了爆赚的 R/R。
4. **去杠杆行情不做 T**：极端杠杆只能用极端下跌化解，去杠杆途中做 T 最大的风险是 put 少赚。
5. **保本止损是铁律**：宁可不赚也别把赢的还回去；杠杆 ETF 打野赚了就换成正股。
6. **fade 极值，不是结构性做空**：基本面没断时对科技战术低配而非战略做空，V 反随时可能来。

## 6. 决策树（把以上拼起来）

```
周一盘前：
  1. 取 GEX 结构 → Spot / HVL / Call Wall / Put Wall / GEX±1D / IVR
  2. Spot vs HVL?
       上方(正gamma) → 默认"回调被买、卖波动、做多为主"，路径受墙压制
       下方(负gamma) → 默认"顺势、买波动、不抄底"，留意自由落体
  3. 仓位结构脆弱性？
       某侧 call/put heavy + 杠杆拉满 → 该侧是受迫平仓的燃料 → 反向埋伏
  4. 选标的：15min RS 扫描 → 最强=龙；标好个股 GEX 位 + 龙气位
  5. 给每条 level 绑动作：上方墙=止盈/做空触发，下方 put=减仓/清仓触发
  6. 风控：先定 invalidation 再定 size；保本止损；去杠杆不做 T
日内：
  价格触及 level 的"一瞬间"动手，不预先挂单等够
```
