> Substack `franktrading.substack.com` 付费周报（2025-10 至今，约 32 篇）、X @Franktradinglog。

## 0. 一句话总纲

> 交易的本质就是：1，你自己不要爆仓。2，发现市场的脆弱结构，尤其是傻钱的运行规律，
> 在他们受迫运动的时候去吃他们。 —— Frank, 2026-05-30 #tradingroom

支撑/阻力**不是画出来的，是受迫资金流（forced flows）的触发位**。一个价位有效，
当且仅当有某类资金在那里被迫买或被迫卖：dealer gamma 对冲、回购盘、CTA、
vol control、杠杆 ETF 再平衡、被动指数盘。没有受迫流锚定的线（如斐波那契）一概不看：
"炒股101：少看鸡毛斐波那契"（04-25）；"据我所知，量化没人看斐波那契"（05-13）。

这条总纲的认识论叫**内生市场理论（Endogenous Market Theory）**：大的行情看似由 news 触发，
实质是由**仓位结构、杠杆分布、波动率传染、被动资金的机械行为**共同决定的。
"事件往往只是点燃引线的火柴，不是炸药本身。"（Substack, 03-08）

## 1. 指数级别：每周一套 GEX/DEX 关键位

每周一开盘前从 GEX dashboard（个股用 Unusual Whales，"看uw就行" 06-06）取一组结构位，
整周围绕它执行。周报固定模板：

| 项目 | 含义 | 用法 |
|---|---|---|
| Spot | 盘前期货位置 | 相对各墙的距离决定本周脚本 |
| **HVL（高波动临界）** | 正/负 gamma 分水岭 | **轴心。上方=正 gamma 稳定区；下方=负 gamma 放大区** |
| Call Resistance（全期限） | 结构性天花板 | 远端目标，Spot 离得远时不参与日内判断 |
| Call Resistance（0DTE） | 日内第一道压力 | 反弹第一目标/止盈位；与全期限重合 → pinning |
| Put Support（结构性/全期限） | 远端防线 | 负 gamma 环境里是"减速带"，不是无脑抄底位 |
| Put Support（0DTE） | 最近的短线防守位 | 守住→反抽；跌破→看下一档 |
| Total GEX/DEX ± 1D 变化 | 结构强化还是退化 | GEX 转负 + 单日大失血 = 结构退化警报 |
| IV30 / IV Rank / HV30 | 波动环境 | 决定用什么结构表达观点（参考 pitfall 19） |

### HVL 规则（核心中的核心）

HVL = High Volatility Level，是做市商 gamma 由正翻负的临界价。它**不是方向预测器，是路径/波动率预测器**——
决定"会不会出现自由落体"，而不是"涨还是跌"。

- **Spot 在 HVL 上方 = 正 gamma 区**：dealer 涨了卖、跌了买 → 波动被压制、回调被吸收、
  "正 gamma 是不可能丝滑暴跌的"（05-30）。这里**做多回调、卖波动**更顺。
- **Spot 在 HVL 下方 = 负 gamma 区**：dealer 越跌越卖、越涨越追 → 趋势自我强化、假突破频发、
  "跌破 hvl 才有自由落体的可能性"（05-30）。这里**顺势、买波动、不要抄底**。
- **实战范式（06-05）**：SPX 失守 7495 HVL 的一瞬间，"正 gamma 翻成 -gamma，做市商被迫越跌越卖"，
  Frank 在**线下方加重空头**（顺势，不是抄底）。
- **反弹按 level 分段推演**：负 gamma 里反弹也很急，但路径是
  `Spot → HVL → 0DTE Call Wall → 全期限 Call Wall` 的分段修复，"不是直接顺滑上攻。
  只有重新站上 7600（0DTE 墙），才有资格讨论 7700（全期限墙）"（06-08）。
- **两墙重合看 pinning**：当 0DTE 与全期限 Call Resistance 同价（如 06-01 的 7600），
  价格容易在该位反复 pin 住、高位震荡消化。

### VIX 决定看 gamma 还是 vanna（重要补充）

> "vix 高的时候，看 vanna 为主；vix 低的时候，看 gamma 为主。"（03-06）
> "左边是 gamma，右边是 vanna"；"vanna 的支撑在 6700""6750 正 vanna，会反弹"（03-06）。

- **低 VIX**：dealer gamma 主导 → 用 HVL/GEX 那套（上面的规则）。
- **高 VIX**：vanna/charm 主导 → 找 **vanna 支撑位**（vol 回落时 dealer 被迫买回的位）。
  "这周的 gamma 和 vanna 环境是支持 squeeze 上去的"（02-23）——高波动里 squeeze 由 vanna 驱动。
- 注意 **VWAP 与 gamma flip 是两个东西**（02-20），但常重合；"啥级别的 vwap 都一样"，
  日内看日线 VWAP，"收不上 vwap 就走"（末日单纪律 03-10）。
- 负 gamma 环境"很容易上下扫"，**开盘不要追空**（02-27）。

### GEX/DEX 结构退化的读法

- Total GEX 仍正 = 舒服的压波动结构；**GEX 转负 = 进入易暴涨暴跌区**。
- 即使 GEX 仍正，但 **GEX/DEX 边际变化（1D Change）为负** = 高位结构在退化，
  进入"高位消化 + Call Wall 博弈"阶段，不是无脑强化。
- DEX 仍正而 GEX 转负 = 中长期仓位没转空、但短期 gamma 缓冲没了 → 最危险的"加杠杆顶部"。

## 2. 个股级别：GEX 位 + 龙气 + 少量经典图形位

> "有些股还有一些独特的支撑位……还有一些重要的技术图形的支撑位，还有个股期权导致的
> gex 位置。这些你都得记脑子里，**提前标在图上**。" —— 06-06

个股 level 的三个来源（按权重）：
1. **个股期权 GEX 位**（看 UW）——同指数逻辑，call wall / put wall（"200 是最大的 call wall" 05-28）。
2. **龙气 / 独特支撑位**——龙头股有自己被资金记住的位（前高、突破回踩位）。
   "APP 昨天突破回踩 500 是真的好机会"（05-27）。
3. **经典图形位**——前高、颈线、趋势线、整数关口。但**仅作辅助**，朴素使用，无指标玄学。

> ⚠️ 杠杆 ETF（SOXL/KORU 等 2x/3x）**不能用正股那套看支撑**：
> "你不能拿 3x etf 看支撑，杠杆 etf 支撑肯定是往下的"（06-10）——它们因衰减天然下移，
> 只能"低买高卖打野"，止损必须挂在**标的指数**上（"上调 koru 止损到大盘 7350" 06-11）。

## 3. 选股：相对强弱（RS）+ 龙气

Level 决定**何时打**，相对强弱决定**打什么**。

> "15min 先看相对强弱，crdo、aaoi、mrvl 最强；之后我预判 7495 止跌，看到 7495 的一瞬间，
> 直接就轰 crdu。" —— 06-05（当日 CRDU +16%）

- **RS 定义**：跌时跌得少、涨时涨得多。"80% 的个股日内走势都是大盘的相似图像，
  但有些股跌的少涨的多，这个就是判断 rs 的重要依据"（06-06）。
- **选股范式（05-27）**："smh 暴跌，这几个货不跌" → 锁定 CRDO/ALAB/ARM（最强 = 龙）。
- **龙气**：最强的那一个就是龙王，下一波重点抄底它（"crdo 是龙王……下周必须重点抄底" 06-05）。

## 4. 受迫资金流玩家清单（"傻钱的运行规律"）

识别"谁在某个位置被迫动"，那个位置就是 level。Frank 的标准清单（05-30 教学）：

| 玩家 | 机械规律 | 可利用的位置/时机 |
|---|---|---|
| **做市商（dealer）** | 按 gamma 对冲 | HVL、call/put wall —— 日内 gamma 支撑阻力的来源 |
| **CTA** | follow trend | 趋势确立后追；近月 CTA 转空 → 不怕被轧空（04-09） |
| **Vol control fund** | 跟踪 RV：RV↓加仓、RV↑减仓 | VIX 急升 → 被迫平高 beta（半导体小票），机械抛售 |
| **Dispersion 盘** | short 指数 vol、long 个股 vol | 管小票；赚 correlation premium；解除时小票剧烈 |
| **杠杆 ETF** | 每日按 NAV 再平衡 | 指数每动 1%→120–180 亿美元被动盘（06-08 周报量化）|
| **回购盘** | 在 VWAP 附近承接 | "拉起来回踩 vwap，回购资金就在 vwap 那买"（05-31）→ 末日轰入 |
| **被动/指数盘** | 权重再平衡 | 单一持股上限触发的强制再平衡（韩国 10% 上限案例 06-08）|

**VWAP = 回购盘的执行位**：知道是谁在 VWAP 买，所以敢在大回购公司（如 NVDA 800 亿回购）
回踩 VWAP 时用 0DTE call 进场。

**仓位结构 = 脆弱性信号**（内生理论的应用）：
- 一边 call-heavy + 杠杆拉满 = 上方脆弱（轧多）；"多头仓位拥挤利空多头"（X 06-10）。
- risk reversal 从 call-skew 翻 put-skew = capitulation tell，可在另一侧建仓（06-08 周报，MU complex）。
- COR1M、单股 vol/VIX 溢价处于历史极值 = 均值回归在即。
- 一句话："谁仓位拥挤就爆谁"（X 06-11）。

## 5. Level 的执行规则（与"怎么找"同样重要）

1. **每条线绑定一个动作，二元化执行**。
   "7500 不破，全力做多，就这一句话"（05-27）；
   "任何时候跌破 spx 7375 减仓，跌破 7350 清仓"（06-07）。线就是 invalidation。
2. **先定线再定仓位**：止损越远，初始仓越小；止损越近，仓位可越大（8 Rules #1）。
3. **卡边缘做不对称赌注**："卡在那个边缘操作——错了基本不会亏钱，对了往往爆赚"（05-30）。
   贴线进场、止损极近 → 小亏大赢的 R/R。
4. **去杠杆行情不做 T**："史诗级的杠杆只能用史诗级的下跌来化解，做 T 最大的风险就是 put 少赚"（06-08）。
5. **保本止损是铁律**："核心就是必须保本止损，不要最后弄亏了"（06-10）；
   杠杆 ETF 打野赚了就去买正股（06-10）。
6. **fade 极值，不是结构性做空**：基本面没断时，对科技保持"战术低配而非战略空头"，
   "V 反随时可能来"（06-08 周报）。

## 6. 决策树（把以上拼起来）

```
周一盘前：
  1. 取 GEX 结构 → Spot / HVL / Call Wall / Put Wall / GEX±1D / IVR
  2. Spot vs HVL?
       上方(正gamma) → 默认"回调被买、卖波动、做多为主"，路径受墙压制
       下方(负gamma) → 默认"顺势、买波动、不抄底"，留意自由落体
  3. 仓位结构脆弱性？
       某侧 call/put heavy + 杠杆拉满 → 该侧是受迫平仓的燃料 → 反向埋伏
  4. 选标的：15min RS 扫描 → 最强=龙；标好个股 GEX 位 + 龙气位
  5. 给每条 level 绑动作：上方墙=止盈/做空触发，下方 put=减仓/清仓触发
  6. 风控：先定 invalidation 再定 size；保本止损；去杠杆不做 T
日内：
  价格到 level 的"一瞬间"动手（如 06-05 看到 7495 即轰 CRDU），不预先挂单等够
```
